Soal 1

Table: Customers

+-------------+---------+
| Column Name | Type    |
+-------------+---------+
| id          | int     |
| name        | varchar |
+-------------+---------+

id adalah primary key untuk tabel ini.
Setiap baris dari tabel ini menunjukkan ID dan nama dari seorang pelanggan.

Table: Orders

+-------------+------+
| Column Name | Type |
+-------------+------+
| id          | int  |
| customerId  | int  |
+-------------+------+

id adalah primary key untuk tabel ini.
customerId adalah foreign key dari ID pada tabel Customers.
Setiap baris dari tabel ini menunjukkan ID dari sebuah pesanan dan ID dari pelanggan yang memesannya.

Tulis solusi untuk menemukan semua pelanggan yang tidak pernah memesan apa pun.

Kembalikan tabel hasil dalam urutan apa pun.

Berikut contoh masukkan dan keluaran.

Input:

Customers table:

+----+-------+
| id | name  |
+----+-------+
| 1  | Joe   |
| 2  | Henry |
| 3  | Sam   |
| 4  | Max   |
+----+-------+

Orders table:

+----+------------+
| id | customerId |
+----+------------+
| 1  | 3          |
| 2  | 1          |
+----+------------+

Output:

+-----------+
| Customers |
+-----------+
| Henry     |
| Max       |
+-----------+

Setiap baris dari tabel ini memberikan informasi tentang nama negara, benua di mana negara tersebut berada, luas wilayahnya, populasi, dan nilai PDB-nya.

Sebuah negara dianggap besar jika:

  • memiliki luas area paling tidak tiga juta (3.000.000 kmÂ²),
  • atau memiliki populasi paling tidak dua puluh lima juta (25.000.000).

Tulislah sebuah solusi untuk menemukan nama, populasi, dan area dari negara-negara besar.

Kembalikan tabel hasil dalam urutan apa pun.

Berikut contoh masukkan dan keluaran.

Input:

World table:

Soal 3

Table: Employee

+-------------+---------+
| Column Name | Type    |
+-------------+---------+
| empId       | int     |
| name        | varchar |
| supervisor  | int     |
| salary      | int     |
+-------------+---------+

empId adalah primary key untuk tabel ini.
Setiap baris dari tabel ini menunjukkan nama dan ID seorang karyawan, gaji mereka, dan ID dari manajer mereka.

Table: Bonus

+-------------+------+
| Column Name | Type |
+-------------+------+
| empId       | int  |
| bonus       | int  |
+-------------+------+

empId adalah primary key untuk tabel ini.
empId juga adalah foreign key ke empId dari tabel Employee.
Setiap baris dari tabel ini berisi ID seorang karyawan dan bonus masing-masing.

Tulislah sebuah solusi untuk melaporkan nama dan jumlah bonus dari setiap karyawan dengan bonus kurang dari 1000.

Kembalikan tabel hasil dalam urutan apa pun.

Berikut contoh masukkan dan keluaran.

Input:

Employee table:

+-------+--------+------------+--------+
| empId | name   | supervisor | salary |
+-------+--------+------------+--------+
| 3     | Brad   | null       | 4000   |
| 1     | John   | 3          | 1000   |
| 2     | Dan    | 3          | 2000   |
| 4     | Thomas | 3          | 4000   |
+-------+--------+------------+--------+

Bonus table:

+-------+-------+
| empId | bonus |
+-------+-------+
| 2     | 500   |
| 4     | 2000  |
+-------+-------+

Output:

+------+-------+
| name | bonus |
+------+-------+
| Brad | null  |
| John | null  |
| Dan  | 500   |
+------+-------+

Penjelasan:
Hanya Brad, John, dan Dan yang memiliki bonus kurang dari 1000

Soal 4

Table: Sales

+-------------+-------+
| Column Name | Type  |
+-------------+-------+
| sale_id     | int   |
| product_id  | int   |
| year        | int   |
| quantity    | int   |
| price       | int   |
+-------------+-------+

(sale_id, year) adalah primary key dari tabel ini.
product_id adalah foreign key ke tabel Product. Setiap baris dari tabel ini menunjukkan penjualan pada produk product_id di tahun tertentu.
Perhatikan bahwa harga adalah per unit.

Table: Product

+--------------+---------+
| Column Name  | Type    |
+--------------+---------+
| product_id   | int     |
| product_name | varchar |
+--------------+---------+

product_id adalah primary key dari tabel ini.
Setiap baris dari tabel ini menunjukkan nama produk dari setiap produk.

Tulis solusi untuk melaporkan product_name, year, dan price untuk setiap sale_id di tabel Sales.

Kembalikan tabel hasil dalam urutan apa pun.

Berikut contoh masukkan dan keluaran.

Input:

Sales table:

+---------+------------+------+----------+-------+
| sale_id | product_id | year | quantity | price |
+---------+------------+------+----------+-------+
| 1       | 100        | 2008 | 10       | 5000  |
| 2       | 100        | 2009 | 12       | 5000  |
| 7       | 200        | 2011 | 15       | 9000  |
+---------+------------+------+----------+-------+

Product table:

+------------+--------------+
| product_id | product_name |
+------------+--------------+
| 100        | Nokia        |
| 200        | Apple        |
| 300        | Samsung      |
+------------+--------------+

Output:

+--------------+-------+-------+
| product_name | year  | price |
+--------------+-------+-------+
| Nokia        | 2008  | 5000  |
| Nokia        | 2009  | 5000  |
| Apple        | 2011  | 9000  |
+--------------+-------+-------+

Penjelasan:
Dari sale_id = 1, kita bisa menyimpulkan bahwa Nokia dijual seharga 5000 pada tahun 2008.
Dari sale_id = 2, kita bisa menyimpulkan bahwa Nokia dijual seharga 5000 pada tahun 2009.
Dari sale_id = 7, kita bisa menyimpulkan bahwa Apple dijual seharga 9000 pada tahun 2011.

| Column Name  | Type    |
